Price_Drop

Live Site | Source Code

Price_Drop was a project built to solve the hassle of keeping up to date with pricing information on your favorite items. This project was my first introduction to web scraping, I used Beautiful Soup in conjunction with Selenium to scrape pricing data off of Target, Farfetch and Etsy. The user is able to make lists to store the saved items so they can always see the lowest prices for them all in one place.
